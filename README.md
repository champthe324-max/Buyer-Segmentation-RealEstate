# 🏠 Buyer Segmentation & Investment Profiling Dashboard

## 📌 Overview
An end-to-end Machine Learning and Data Analytics project that segments real estate buyers based on demographics, investment behavior, and purchasing patterns.

This project includes an interactive Streamlit dashboard with dynamic filters, KPI cards, business insights, and data visualizations.

## 🚀 Features
- Interactive Streamlit Dashboard
- Buyer Segmentation using K-Means Clustering
- Dynamic Filters
- KPI Metrics
- Interactive Plotly Charts
- Business Insights
- Download Filtered Dataset

## 📊 Dashboard Modules
- Buyer Persona Distribution
- Loan Applied Analysis
- Buyers by Country
- Buyers by Region
- Investment Analysis
- Satisfaction Analysis
- Referral Channel Analysis
- Age Distribution

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Streamlit
- Git & GitHub
